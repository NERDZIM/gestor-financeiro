Seguranca e Privacidade

Ola, usuario

Saiba como seus dados sao protegidos nesta aplicacao.

  • Sem CDN na interface

    CSS e JavaScript ficam no proprio projeto, reduzindo risco de script externo roubar dados.

  • Protecoes HTTP ativas

    CSP, anti-clickjacking, politica de referencia, CSRF e cookies HTTPOnly/SameSite.

  • Isolamento por usuario

    Toda consulta de transacao passa pelo usuario autenticado antes de listar, editar ou excluir.
